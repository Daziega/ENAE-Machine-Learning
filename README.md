# ENAE-Machine-Learning
Machine Learning exercises from class 

**FINAL CASE**
As a final case for the module, students will apply the concepts learned to a real Lending Club dataset , which contains information about loans issued through its platform. 

The main objective of the exercise is to  predict whether a loan will be fully paid or will default. To achieve this, students must apply and compare different machine learning models.

The final results must be presented through:

A clear comparison of the models used (3 models).

A ranking of the models according to different performance metrics, such as:

AUC

Precision

Recall

F1-score

Other relevant metrics when appropriate

The solutions must be uploaded to the platform in .ipynb format (IPython Notebook) and must obligatorily include all the Python code necessary to reproduce the analysis and obtain the final results.
